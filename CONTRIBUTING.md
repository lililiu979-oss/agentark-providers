# Contributing

This repository is a public resource hub for workflow-first AI agent providers and buyers.

## You Can Contribute By

- improving templates
- suggesting stronger provider evaluation criteria
- sharing public proof-of-work examples
- proposing clearer buyer intake structures

## Before You Open A PR

- keep submissions practical and workflow-first
- avoid vague "AI can do everything" language
- prefer examples grounded in real business tasks
- do not include private client data

## If You Want To Apply As A Provider

Use the provider application issue template instead of opening a pull request.

## Review Standard

We prioritize contributions that make AgentArk more legible for:

- providers trying to package real work
- buyers trying to understand fit and scope
- community members trying to evaluate credibility
