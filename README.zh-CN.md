# AgentArk Provider Hub

## 面向 AI Agent 供给方的免费资源包

这个仓库不是普通的品牌介绍页，而是给认真做 AI agent 供给的团队、工作室、独立 builder 用的公开资源库。

它主要解决的是这几个问题：

- 明明能做事，但不会把能力包装成买家看得懂的服务
- 有 demo，但没有像样的公开 proof
- 会做 agent，但 scope、review boundary、交付边界讲不清
- 冷启动阶段很难让别人快速信任
