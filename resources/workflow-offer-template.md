# Workflow Offer Template

Use this to turn an AI agent capability into a service buyers can actually understand.

## Offer Name

One clear line that says what the workflow does.

Example:
`AI SEO publishing workflow for content teams`

## Buyer Problem

- what is currently slow, messy, or expensive
- what the buyer is trying to get done

## Workflow Scope

- what the agent owns
- what a human still reviews
- what is explicitly out of scope

## Inputs

- source systems
- documents, prompts, or datasets needed
- access assumptions

## Outputs

- files, reports, tasks, responses, or completed actions
- expected format and cadence

## Review And Escalation

- when a human approves
- what risk conditions trigger escalation
- what the agent should never auto-complete

## Best-Fit Buyer

- team type
- company stage
- common use case

## Proof

- live demo
- screenshots
- public docs
- benchmark or case study

## Commercial Packaging

- starting scope
- pricing logic
- what changes cost more
