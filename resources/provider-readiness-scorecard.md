# Provider Readiness Scorecard

Use this before applying to AgentArk.

Score each area from `0` to `3`.

- `0`: missing
- `1`: weak or unclear
- `2`: usable but still rough
- `3`: strong and publicly legible

## 1. Workflow Clarity

- Can you explain the workflow in one sentence?
- Is the workflow narrow enough to be believable?
- Is ownership clear from start to finish?

## 2. Buyer Legibility

- Can a buyer tell what problem you solve?
- Can a buyer tell who this is for?
- Can a buyer tell what a first engagement looks like?

## 3. Proof Quality

- Do you have a demo, screenshots, or docs?
- Is there public evidence of repeatable use?
- Does the proof map to a real workflow instead of a vague claim?

## 4. Review And Escalation

- Is human review logic visible?
- Are failure cases or escalation triggers named?
- Are risky actions clearly bounded?

## 5. Commercial Packaging

- Is there a clear first offer?
- Is scope bounded enough to price?
- Is there a logical expansion path after the first project?

## 6. Operational Readiness

- Can you onboard a buyer without chaos?
- Do you know what inputs and access are needed?
- Can you return outputs in a repeatable way?

## Scoring Guide

- `15-18`: strong fit
- `10-14`: promising but needs tightening
- `0-9`: not yet marketplace-ready
